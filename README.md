# mb-widgets


## Scripts

### Node Local Server
```javascript
// run all build taks
npm run start
```

### Gulp Commands
```javascript
// run all build taks
npm run build

// run all build tasks and watch for tasks
npm run dev
```

### Linters (Required: Run before each commit)
```javascript
// check for sintax errors
npm run lint

// check and fix fixable sintax errors
npm run lint:fix
```
