# mb-widgets


## Scripts

### Node Local Server
```javascript
// run all build taks
npm run start
```

### Gulp Commands
```javascript
// run all build taks
npm run build

// run all build tasks and watch for tasks
npm run dev
```

### Before each commit
```javascript
// check sintax errors
npm run lint

// fix fixable sintax errors
npm run lint:fix
```

### Linter: [Github Super Linter](https://github.com/github/super-linter#supported-linters)
